Build a GTM Cockpit web application for Fideltour, a B2B SaaS company 
specializing in CRM and loyalty solutions for the hospitality industry. 
This is Platform 1 (MQL/GTM) - focused on lead generation, enrichment, 
and nurturing until leads become SQLs (Contact Value ≥ 50).

## TECH STACK
- React + TypeScript (frontend)
- Tailwind CSS (styling)
- React Router (navigation)
- Recharts (charts and analytics)
- Lucide React (icons)
- Mock data (no backend yet - all data hardcoded as realistic examples)

## DESIGN SYSTEM
- Dark theme: background #0F1117, cards #1A1D27, borders #2A2D3E
- Primary accent: #6366F1 (indigo)
- Success: #10B981 (green)
- Warning: #F59E0B (amber)  
- Danger: #EF4444 (red)
- Text primary: #F9FAFB, Text secondary: #9CA3AF
- Font: Inter
- Rounded corners: rounded-xl for cards, rounded-lg for buttons
- Sidebar navigation (left), main content (right)

## APPLICATION STRUCTURE
6 main screens accessible via left sidebar:

1. Dashboard (home icon) - "GTM Dashboard"
2. Database (database icon) - "Hotel Database"  
3. Lead Detail (user icon) - "Lead Profile" (accessible by clicking a hotel)
4. Sequences (git-branch icon) - "Cadences"
5. Unibox (inbox icon) - "Unified Inbox"
6. Analytics (bar-chart icon) - "Analytics"

## NAVIGATION & UX

- Sidebar: fixed left, 240px wide, shows icon + label
- Active state: indigo background on sidebar item
- All screen transitions: instant (no animation needed for MVP)
- Clicking hotel in database → goes to Lead Profile screen
- Clicking back arrow in Lead Profile → returns to database
- Clicking conversation in Unibox → opens detail in right panel
- Responsive: desktop only (min-width 1280px)
- No authentication needed (direct app access)

## IMPORTANT NOTES

- All data is mock/hardcoded - no API calls
- Focus on visual completeness over functionality
- Every screen should look production-ready
- Use realistic numbers that make business sense for a hotel SaaS
- Spanish language for all UI labels and content
- The app name in the sidebar header: "GTM Cockpit" with Fideltour logo 
  placeholder (use a simple indigo rounded square with "F" letter)
